thread

類別繼承thread。

class myThread(threading.Thread):
    def __init__(self , threadID, name, counter):
        threading.Thread.__init__(self)
        self.ThreadID = threadID
        self.name = name
        self.counter = counter
    
    def run(self):
        print(f'start : {self.name} --')
        sleep_count(self.name, self.counter, 3)
        print(f'end : {self.name} --')

def sleep_count(name ,delay ,counter ):
    while(counter):
        time.sleep(delay)
        print(f'{name} : {counter} -- {time.strftime("%H:%M:%S", time.localtime())}')
        counter -= 1

start()

開始執行多線程，若單獨呼叫類別中函式，則會變單線程

如下範例

thread1.run()
thread2.run()

"""
start : dwvwdv --
dwvwdv : 3 -- 23:24:24
dwvwdv : 2 -- 23:24:25
dwvwdv : 1 -- 23:24:26
end : dwvwdv --
start : xiao --
xiao : 3 -- 23:24:28
xiao : 2 -- 23:24:30
xiao : 1 -- 23:24:32
end : xiao --
"""

而使用start()後，才會同時進行

thread1.start()
thread2.start()

"""
start : dwvwdv --
start : xiao --
dwvwdv : 3 -- 23:26:58
xiao : 3 -- 23:26:59
dwvwdv : 2 -- 23:26:59
dwvwdv : 1 -- 23:27:00
end : dwvwdv --
xiao : 2 -- 23:27:01
xiao : 1 -- 23:27:03
end : xiao --
"""

join()

子程序執行結束後關閉，在接下來需要子程序資料時使用，等待子程序運算完後的數據，或單純用來正常關閉子程序。

thread1.join()
thread2.join()

threading.get_ident()

可以得到該線程的的編號。

ident = threading.get_ident()

setDaemon()

設置守護線程，當主程式結束後一併結束該線程。

thd = threading.Thread(target = task)
thd = setDaemon(True)
thd.start()

ThreadPoolExecutor(int )

線程池，使用在需要多個thread時，一般的創建若是頻繁，則創造、銷毀線程過程非常消耗資源。

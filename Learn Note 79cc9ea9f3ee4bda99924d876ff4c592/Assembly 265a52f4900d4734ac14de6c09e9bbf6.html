簡易理解暫存器

  • rax、rbx、rcx、rdx、rsi、rdi、rbp、rsp、r8~r15，可以看做都是一樣的東西，arm架構則是r0到r31
  • rip則是存放當前指令的位址
  • cs、ss、ds、es，不必糾結，歷史性的東西
  • fs、gs較為常用
  • eflgs 與JCC指令配套，JCC就是用來跳轉的

雖說rax到r15都是一樣的，但現在約定成俗的狀況是，rbp與rsp管控靜態記憶體(棧內存)，而rax則用來接收return的值。

Assembly Code

常見指令

mov、add、sub、lea、push、pop、cmp、jmp、jcc、call、leave、ret

大部分都很直觀好理解，其中值得拿出來講的有以下幾個：

lea

全名Load effective address，以前用來載入地址，現在用來做計算，例如 +8 +5等等，他可以不用改變後者的值，而且指令短，不用先sub後再mov

cmp

類似sub指令，舉個例子

sub a1,0x20 → a1 = a1-0x20

cmp a1,0x20 → a1-0x20

cmp不存取結果，而將a1-0x20給之後的JCC進行判斷怎麼跳轉

test

test跟上面cmp有異曲同工之妙，對比的是and

and eax,eax → eax = eax&eax

test eax,eax → eax&eax

假設是像上面這樣，這個指令可以直接看作是 cmp eax,0，eax是不是等於0

jcc跳轉系列

預設邊譯、nopie、static

三者特點

nc 127.0.0.1 [port]

沒問題就直接調用腳本連接、調試或載入payload

import socket
import telnetlib
import struct

def P32(val):
    return struct.pack("",val)

def pwn():
    s = socket.socket(socket.AF_INET,socket.SOCK_STREAM)
    s.connect(("127.0.0.1", 8866))
    #system_addr = P32(0x080483A0)
    #binsh_addr = P32(0x08048610)
    payload = 'A'*8 +'a'
    s.sendall(payload.encode())
    t = telnetlib.Telnet()
    t.sock = s
    t.interact()

if __name__ == '__main__':
    pwn()

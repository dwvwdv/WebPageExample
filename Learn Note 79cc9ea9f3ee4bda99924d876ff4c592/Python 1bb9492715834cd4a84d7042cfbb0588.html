可變參數

*args

將傳入參數收集至Tuple

def fun(*args):
    for i in args:
        print(i)
    print(type(args))

nums = [1,3,5,7,9]
fun(nums)

"""
output:

[1,3,5,7,9]
<class 'tuple'>
"""

**kwargs

傳入字典參數

param = {'sep': ' #','end':'666\n'}
print('this is test text.',**param)

"""
output:

this is test text.666
"""

執行

python.exe與pythonw.exe

*.py檔使用 python.exe執行，*.pyw檔使用pythonw.exe，差別在於pythonw.exe執行後不會產生dos視窗，是為了實現介面化的。

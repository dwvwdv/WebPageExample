Camera

Clare flags

  • SkyBox
  • Solid Color
  • Death only
  • Don’t Clear

鏡頭看見的天空、地面狀況選擇。

Death only 空白部分不渲染。

Don’t Clear 容易花屏，基本不用。

Culling Mask

鏡頭可以選擇屏蔽一個或多個 層級的所有物件。

Projection

  • Perspective
  • Orhographic

前者為3D預設正常視角，後者忽略遠近，2D視角。

鏡頭縮放

  • FOV
  • Clipping Planes

FOV為視角倍率、Clipping Planes為可視距離。

分屏調整

  • Viewport Rect
  • Depth

Viewport Rect可用來切屏，多個鏡頭播放，畫面重疊有重疊部分設置Depth層級可確保正常顯示。

Skybox 天空盒

包裝整個場景的景色，將整個場景包起來的一個Material。

  • 6 Sided
  • Procedural
  • Cubemap 用到極少

使用方式：

第一種

攝像機添加Skybox Component。

第二種

Window → Rendering → Lighting → SkyBox調用材質。

Global Illumination

  • GI，全局光照
  • 能計算直接光、間接光、環境光、反射光的光照系統
  • GI算法可以使渲染光照看起來更真實

直接光 - Light Component取得

  • Directional - 平行光，高低無關，照射場景所有物體，模擬太陽
  • Point - 點光源，類似燈泡的光，照射範圍是圓的
  • Spot - 聚光燈，手電筒光
  • Area - 從一個面發射光線，只照射該區物體，僅烘培時有效，用在光線較為集中區域

間接光

其他光造成的反射光照到物體上

反射光

其他光造成的反射光

Window → Rendering → Lighting → Environment → Reflection

環境光

場景中所有物體的光照

Window → Rendering → Lighting → Environment

實時GI

運行期間更改光源，可立即更新

烘焙

  1. 將場景靜態物品標為static
  1. Environment → Generate Lighting

  • Backed Resolution 光照貼圖分辨率，建議比實時GI高10倍
  • Backed padding 光照貼圖形狀間距，取2到200
  • Compressed 壓縮，可提高性能，縮小容量，但畫質降低

光源偵測

烘焙的結果不是真實的，會導致物體與場景光線無法融合，非常不真實，可以利用Light Probes組件通過Probe收集光影信息，對鄰近幾個Probe進行插值運算，把光照作用到物體上

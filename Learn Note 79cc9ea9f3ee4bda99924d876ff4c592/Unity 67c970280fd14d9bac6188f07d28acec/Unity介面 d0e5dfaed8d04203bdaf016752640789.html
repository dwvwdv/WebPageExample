Scenes面板

功能

遊戲場景布置、創作

Hot Key

滑鼠右鍵+Alt : 視角放大縮小

滑鼠左鍵+Alt : 視角總是面向物件

選取物件+F : 視角調到正中心

選取物件+Shift+F : 視角追蹤移動物件

滑鼠右鍵+W/A/S/D Q/E : 視角前後左右移動 、 上下移動

Q/W/E/R/T : 切換滑鼠功能

選取物件+V : 與另一個物件貼合

Ctrl+ P : 執行

ISO與Persp視角

ISO視角較為2D，適合用來做2D場景。

Persp視角較為立體，在觀看XYZ軸變化更有效率。

Hierarchy面板

功能

場景物件管理

遊戲物件(GameObject)

遊戲場景中的物體，能繼承屬性，物件與物件可以有父子關係，是一種容器，掛載組件。

過濾器與渲染器聯合使用，才能讓模型顯示到螢幕上。

快捷鍵

Ctrl+D : 複製當前選擇物件。

Game面板

功能

實際遊戲畫面觀看視角，方便調整鏡頭。

調整各個螢幕像素，如 1920*768等等，應對各種螢幕。

Project面板

功能

管理專案的所有檔案。

Models檔案

使用素材存放位置。

Scenes檔案

存放所有場景使用物件與物件位置，場景儲存下來就是Scenes檔，通常一個關卡就是一個場景。

Inspector面板

功能

管理組件(Component)的參數

組件(Component)

遊戲物件(GameObject)的功能模組，每個組件都是一個類的實作，可以當作屬性來看

Transform 變換組件: 決定物件的位置、大小、旋轉比

Mesh Filter 網格過濾器 : 從資源中取得網格訊息，改變物件形狀

Mesh Renderer 網格渲染器 : 從 Mesh Fileter 取得幾何形狀，再根據 Transhform 定義位置進行渲染
